## Image Requirements:
- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 1200x750px (16:10 aspect ratio)
- **Max File Size**: 500KB per image (optimize for web)
- **Quality**: High-quality screenshots or certificate images

## Tips:
1. Take screenshots of your actual certificates
2. Use tools like TinyPNG or Squoosh to compress images
3. Keep aspect ratio consistent for better visual appearance
4. Use descriptive names matching the ones listed above
5. If you have certificate PDFs, convert them to images using a PDF-to-image tool

## Fallback:
If you don't have an image for a certificate, the page will show a placeholder with an external link icon.
